# Physical AI & Humanoid Robotics Textbook

A comprehensive textbook on Physical AI and Humanoid Robotics, designed for advanced learners who want to understand and develop humanoid robots with advanced AI capabilities.

## Overview

This textbook represents the cutting edge of embodied artificial intelligence, where digital intelligence meets physical reality through advanced humanoid robotic systems. It covers four comprehensive modules:

- **Module 1**: The Robotic Nervous System (ROS 2)
- **Module 2**: The Digital Twin (Gazebo & Unity)

## Local Development

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions when changes are pushed to the `main` branch.
