# Module 1: The Robotic Nervous System (ROS 2)

This module covers the foundational middleware for robot control using ROS 2.

## Learning Objectives
- Understand ROS 2 architecture and DDS middleware
- Learn to create and connect ROS 2 nodes using rclpy
- Master launch files and parameter management
- Work with URDF for humanoid robot models

## Topics
- ROS 2 architecture and DDS middleware
- Nodes, topics, services, and actions
- Python-based ROS 2 development using rclpy
- Launch files and parameter management
- URDF (Unified Robot Description Format) for humanoid robots
